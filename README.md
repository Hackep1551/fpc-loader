# FPC Plugin Loader

Плагин для FunPayCardinal. Загружает зашифрованные плагины с лицензионного сервера.

## Установка

1. Скопируйте `PluginLoader.py` и `loader_core.so` (Linux) / `loader_core.dll` (Windows) в папку `plugins/`
2. Перезапустите FunPayCardinal
3. Введите в боте Cardinal: `/lkey ВАШ-UUID`

UUID получить у @kapystus
